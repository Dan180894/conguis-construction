Congui's Construction — Sitio Web
Sitio web profesional desarrollado para Congui's Construction, empresa de construcción y remodelación ubicada en Guanacaste, Costa Rica. Proyecto real, desplegado en producción.
Ver sitio en vivo

Características

Multilingüe — Sistema de internacionalización (i18n) en español, inglés y francés, desarrollado desde cero en JavaScript vanilla sin librerías externas
Responsive — Diseño adaptable a móvil, tablet y escritorio
Animaciones — Scroll reveal con IntersectionObserver y transiciones CSS
Integración WhatsApp — Botones de contacto directo con mensajes predefinidos por idioma
SEO básico — Meta tags, títulos descriptivos y texto alternativo en imágenes
Sin dependencias — HTML, CSS y JavaScript puros, sin frameworks ni librerías

Lo que aprendí / apliqué

Implementar un sistema de traducción dinámica sin librerías usando objetos JavaScript y data-i18n
Usar IntersectionObserver para animaciones de entrada al hacer scroll
Diseñar layouts complejos con CSS Grid y Flexbox
Aplicar variables CSS (custom properties) para mantener consistencia visual
